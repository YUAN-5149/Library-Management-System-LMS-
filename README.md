# 溫書閣 Library System

一套為消防局設計的圖書館管理系統，以單一 HTML 檔案運行，無需安裝任何後端環境。

## 🚀 快速開始

直接用瀏覽器開啟 `index.html` 即可使用，無需 Node.js、Python 或任何伺服器。

## 🔐 管理員登入

- 帳號：`8200`
- 密碼：`8200`

> 建議上線前於原始碼中修改 `const AID` 與 `const APW` 的值。

## 📋 功能清單

### 一般功能（讀者）
| 功能 | 說明 |
|------|------|
| 館藏總覽 | 統計儀表板：館藏數量、熱門書目、分類分佈、最新入館、評分榜 |
| 書目管理 | 搜尋、分類篩選、書況篩選、CSV 匯出、書目詳情 |
| 電子書目 | 電子書卡片瀏覽 |
| 分類管理 | 20 個分類的館藏統計，點擊快速篩選 |
| 新書上架 | 依入館日期排序，標示本週/近 30 天新書 |
| 隨機推薦 | 每次隨機抽出 3 本實體書 + 1 本電子書 |
| 我的紀錄 | 輸入姓名或電話查詢個人借閱、歸還歷史與預約清單 |
| 書目評分 | 1–5 顆星評分 + 一句話短評 |

### 管理員專區
| 功能 | 說明 |
|------|------|
| 借閱管理 | 借出中 / 已歸還 / 預約清單三分頁，支援歸還操作 |
| 到期提醒 | 逾期 / 3 天內 / 7 天內分類統計，色彩高亮 |
| 月度報表 | 月份選擇器、四卡統計、熱門書目 Top 5、熱門分類、借閱人排行、明細表格，可列印/匯出 PDF |
| 新增書目 | 實體書 / 電子書，含封面圖片上傳（base64） |
| 批量匯入 | CSV 上傳，支援中英文欄位名稱，預覽後確認匯入 |
| QR 產生器 | 每本書產生 QR Code，可列印單張標籤 |
| 書目編輯 | 修改所有書目欄位，含封面更換 |
| 複本管理 | 多冊追蹤，借閱時可選擇冊別 |
| 借閱上限 | ＋/－ 設定每位會員最多借閱本數（1–20） |
| 會員管理 | 新增會員，含借閱進度條與上限警示 |

### 體驗功能
| 功能 | 說明 |
|------|------|
| 暗色模式 | 側邊欄底部一鍵切換，所有介面區域支援 |
| 書目詳情 | 封面、分類、複本管理、評分、同分類相關推薦 |

## 🛠 技術架構

- **前端框架**：React 18（CDN，瀏覽器端 Babel 編譯）
- **樣式**：CSS-in-JS（inline style）
- **字體**：Playfair Display、Noto Serif TC（Google Fonts）
- **QR Code**：qrcodejs（Cloudflare CDN）
- **資料儲存**：瀏覽器 state（無持久化，重整後重置）

## ⚠️ 注意事項

- 資料僅存在瀏覽器記憶體中，重新整理後會重置為預設資料
- 如需持久化，可考慮串接 Google Sheets（程式碼內有 `SID` 設定預留）
- 封面圖片以 base64 格式儲存於 state，大量上傳可能影響效能
